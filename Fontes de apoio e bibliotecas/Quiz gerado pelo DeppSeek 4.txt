Nível Fácil
O que é Gestão de Riscos?
a) Um processo para eliminar todos os riscos de um projeto.
b) Um processo de identificação, análise e resposta aos riscos.
c) Uma técnica para aumentar os riscos em um projeto.
Resposta correta: b) Um processo de identificação, análise e resposta aos riscos.

Qual é o primeiro passo no processo de Gestão de Riscos?
a) Análise de riscos.
b) Identificação de riscos.
c) Planejamento de respostas aos riscos.
Resposta correta: b) Identificação de riscos.

O que é um risco positivo?
a) Um risco que não afeta o projeto.
b) Um risco que pode trazer benefícios ao projeto.
c) Um risco que sempre causa prejuízos.
Resposta correta: b) Um risco que pode trazer benefícios ao projeto.

Qual é o objetivo principal da Gestão de Riscos?
a) Eliminar todos os riscos.
b) Minimizar o impacto negativo e maximizar oportunidades.
c) Ignorar os riscos e focar apenas no planejamento.
Resposta correta: b) Minimizar o impacto negativo e maximizar oportunidades.

O que é uma matriz de probabilidade e impacto?
a) Uma ferramenta para priorizar riscos.
b) Um método para eliminar riscos.
c) Uma técnica para aumentar os riscos.
Resposta correta: a) Uma ferramenta para priorizar riscos.

Qual é a diferença entre risco e problema?
a) Risco é algo que pode acontecer; problema é algo que já aconteceu.
b) Risco e problema são a mesma coisa.
c) Problema é algo que pode acontecer; risco é algo que já aconteceu.
Resposta correta: a) Risco é algo que pode acontecer; problema é algo que já aconteceu.

O que é um plano de contingência?
a) Um plano para evitar riscos.
b) Um plano para lidar com riscos que se materializam.
c) Um plano para aumentar os riscos.
Resposta correta: b) Um plano para lidar com riscos que se materializam.

Qual é a principal função de um gerente de riscos?
a) Ignorar os riscos.
b) Gerenciar e mitigar os riscos do projeto.
c) Aumentar os riscos para desafiar a equipe.
Resposta correta: b) Gerenciar e mitigar os riscos do projeto.

O que é mitigação de riscos?
a) Eliminar completamente os riscos.
b) Reduzir a probabilidade ou impacto de um risco.
c) Ignorar os riscos.
Resposta correta: b) Reduzir a probabilidade ou impacto de um risco.

O que é um registro de riscos?
a) Um documento que lista todos os riscos identificados.
b) Um método para eliminar riscos.
c) Uma técnica para aumentar os riscos.
Resposta correta: a) Um documento que lista todos os riscos identificados.

Resposta correta: a) Análise SWOT.

b) Uma análise que prioriza riscos com base em sua probabilidade e impacto.
c) Uma análise que ignora os riscos.
Resposta correta: b) Uma análise que prioriza riscos com base em sua probabilidade e impacto.

Resposta correta: a) Uma análise que usa dados numéricos para avaliar riscos.
