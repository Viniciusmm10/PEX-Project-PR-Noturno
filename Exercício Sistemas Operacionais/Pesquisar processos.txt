csrss.exe - Julio
O csrss.exe (Client Server Runtime Process) é um processo essencial do Windows responsável por funções críticas do sistema operacional, como a criação e o encerramento de threads e a gestão de janelas em versões mais antigas do Windows. Ele faz parte da arquitetura do subsistema do Windows e roda em segundo plano como um processo protegido.

Principais características do csrss.exe:
	•	Está localizado na pasta C:\Windows\System32.
	•	Executa tarefas relacionadas à interface gráfica e à criação de processos.
	•	Sempre é iniciado com o sistema operacional e não pode ser encerrado sem causar uma falha grave no Windows (tela azul).

svchost.exe - Vitor
"svchost.exe" é a abreviação de "Service Host" (Host de Serviço). É um processo genérico que o Windows utiliza para executar vários serviços.
Em vez de cada serviço rodar em seu próprio processo separado, o Windows os agrupa sob o "svchost.exe" para conservar os recursos do sistema.

omtsreco.exe - Samuel
O processo omtsreco.exe no Windows é parte do Oracle MTS Recovery Service, desenvolvido pela Oracle Corporation. Este serviço é um componente crucial do software de banco de dados da Oracle, responsável por gerenciar transações distribuídas. Ele garante que as transações sejam concluídas com sucesso ou revertidas corretamente em caso de falha, mantendo a integridade dos dados no banco de dados

Video.UI.exe - Vinicius
O Video_UI.exe é um processo relacionado a interfaces de usuário (UI) de programas de vídeo. Ele pode ser parte de um software de reprodução de mídia ou de edição de vídeo, como o VLC, Windows Media Player, ou algum outro programa de vídeo instalado na sua máquina.
Esse processo provavelmente está em andamento porque você tem um programa que lida com vídeo aberto ou em execução, como um reprodutor de mídia, editor ou até um software que manipula vídeos de alguma maneira. O processo é responsável por exibir a interface gráfica e permitir a interação do usuário, como controlar a reprodução ou editar vídeos.
Se você não tem um programa de vídeo ativo e o processo está em execução, pode ser útil investigar mais a fundo para garantir que não seja um programa indesejado ou malicioso.
